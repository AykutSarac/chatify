
# Chatify

Full stack chat application made with Vue & Fastify.

**Frontend Stack:**

* Vue w/ Composition API

* Bootstrap

* Vite

**Backend Stack:**

* Fastify

* JSON-Server

## Installation

```bash
// Install the required packages
npm install

// Run json-server/fastify & vue client
npm run concurrently
```
