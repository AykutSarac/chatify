## Preview

![preview](https://i.ibb.co/RybqXnF/localhost-3000.png)

**Frontend Stack:**

- Vue 3 & Composition API

- Bootstrap

- Vite

**Backend Stack:**

- Fastify

- JSON-Server

## Installation

```bash
// Install the required packages
npm install

// Run json-server/fastify & vue client
npm run concurrently

>> Chatify running at http://localhost:3000
```
